Setting up a server

What you'll need

Node installed on your machine. You won't need to install any additional modules; we'll be using built in modules for this tutorial.

Modules you'll need

Declare the following variables:

  1. an http variable with a call to require the html module.
  2. a port variable. Set this to process.env.PORT || 8080
  3. an fs variable. fs is a built in file system on node. It will allow you to make requests from your browser for files such as stylesheets
